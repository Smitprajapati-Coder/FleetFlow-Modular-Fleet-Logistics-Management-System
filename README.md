# 🚚 FleetFlow – Fleet & Logistics Management System

FleetFlow is a full-stack web application designed to help organizations manage their vehicle fleets efficiently.  
It provides real-time CRUD operations, search, filtering, and dashboard features using modern web technologies.

---

## 📌 Features

- ✅ Vehicle Management (Add / Edit / Delete / View)
- 🔍 Search and Filter Vehicles
- 📊 Dashboard Interface
- 🔗 REST API Integration
- 💾 SQLite Database
- ⚡ Real-Time UI Updates
- 🔐 Basic Authentication (Optional)

---

## 🛠️ Tech Stack

### Frontend
- React (Vite)
- Axios
- HTML / CSS / JavaScript

### Backend
- Node.js
- Express.js
- Sequelize ORM
- SQLite Database

## 🚀 Getting Started

Follow these steps to run the project locally.

---

### ✅ Prerequisites

- Node.js (v16+ recommended)
- npm

---

### ▶️ Backend Setup

```bash
cd backend
npm install
node server.js
````

Server runs at:

```
http://localhost:5000
```

---

### ▶️ Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend runs at:

```
http://localhost:5173
```

## 📡 API Endpoints

| Method | Endpoint      | Description      |
| ------ | ------------- | ---------------- |
| GET    | /vehicles     | Get all vehicles |
| POST   | /vehicles     | Add new vehicle  |
| PUT    | /vehicles/:id | Update vehicle   |
| DELETE | /vehicles/:id | Delete vehicle   |

## 🧩 System Architecture

```
User
 ↓
React Frontend
 ↓ REST API
Node + Express Backend
 ↓
SQLite Database
```

The frontend communicates with the backend through REST APIs.
The backend processes requests and interacts with the database.
